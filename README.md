**미로찾기 게임**


1. 환경 설정 및 기초 개념 (windows.h)  
 API 활용: 콘솔 출력 제어  
  방향키 입력 시 콘솔 조작  
  화면 업데이트  
  화면 색상 변경  
  기존 좌표 삭제  
  좌표 제어  
  커서 이동 함수  
  현재 커서 위치 저장


2. 플레이어 이동 로직  
   입력 감지 (상하좌우 구분 등)  
  상호작용 및 이벤트 처리  
  기본 이동 (빈칸으로 이동)  
  벽(#) : 이동 X  
  함정(^) : hp - 10  
  체력 회복(a) : hp 를 100으로 조정  
  몬스터(M) : 즉시 게임 종료 (패배) --> 적절한 출력  
  목적지(Y) : 즉시 게임 종료 (승리) --> 적절한 출력  
 

3. 추가 기능  
  상태창 (hp : 100) 출력  
  승리/패배 출력: 화면 색상 변경
